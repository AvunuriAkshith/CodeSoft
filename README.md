# ATM Interface

## Overview

ATM Interface is a Java-based console application that simulates the basic functionalities of an Automated Teller Machine (ATM). The project demonstrates Object-Oriented Programming concepts such as classes, objects, encapsulation, and method interactions.

## Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Input Validation
* Menu-Driven Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class

## Project Structure

ATMInterface/
├── BankAccount.java
├── ATM.java
└── Main.java

## How to Run

1. Compile the project:

javac *.java

2. Run the application:

java Main

## Learning Outcomes

* Understanding OOP principles
* Working with classes and objects
* Implementing menu-driven applications
* Handling user input and validations
