# Currency Converter

## Overview

Currency Converter is a Java application that converts an amount from one currency to another. The project demonstrates Java fundamentals, conditional logic, user input handling, and modular programming.

## Features

* Currency Conversion
* Multiple Currency Support
* User-Friendly Console Interface
* Error Handling for Unsupported Currency Pairs

## Technologies Used

* Java
* OOP Concepts
* Scanner Class

## Project Structure

CurrencyConverter/
├── CurrencyConverter.java
└── Main.java

## How to Run

1. Compile the project:

javac *.java

2. Run the application:

java Main

## Sample Usage

Enter Amount: 100
From Currency: USD
To Currency: INR

100 USD = 8350 INR
