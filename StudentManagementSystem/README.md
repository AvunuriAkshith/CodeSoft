# Student Management System

## Overview

Student Management System is a Java-based console application that allows users to manage student records efficiently. It provides CRUD (Create, Read, Update, Delete) operations and demonstrates the use of collections and Object-Oriented Programming principles.

## Features

* Add Student
* View All Students
* Search Student by Roll Number
* Update Student Information
* Delete Student Record
* Menu-Driven Interface

## Technologies Used

* Java
* OOP Concepts
* ArrayList Collection Framework
* Scanner Class

## Project Structure

StudentManagementSystem/
├── Student.java
├── StudentManagementSystem.java
└── Main.java

## How to Run

1. Compile the project:

javac *.java

2. Run the application:

java Main
